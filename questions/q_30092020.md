## Question:
What would the following return in R?
```r
1 == "1"
```

## Topic:
1. Programming
2. R
3. Vectors

## Options:
1. TRUE
2. FALSE
3. Error
4. NULL

## Correct Option:
1. TRUE

## Explanation:
Due to coercion rule, the `1` on LHS will be coerced to `"1"`, hence we will get TRUE.
