## Question:
In R the following would produce?
```r
-4^2
```

## Topic:
1. R Programming
2. Operator presedence

## Options:
1. 16
2. -16
3. Error
4. NULL

## Correct Option:
2. -16

## Explanation:
`^` has higher precedence than `-`. So the above example is like `-(4^2)`.
